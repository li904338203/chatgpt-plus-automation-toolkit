# Runtime Package

The ready-to-run sanitized runtime is committed directly at:

```text
ChatGPTAssistantPanel\ChatGPTAssistantPanel.exe
```

Before running, edit `ChatGPTAssistantPanel\.env`, `ChatGPTAssistantPanel\config.yaml`, and put your own pools under `ChatGPTAssistantPanel\data\`.

No private account, proxy, card, phone, API key, or local runtime data is included.
